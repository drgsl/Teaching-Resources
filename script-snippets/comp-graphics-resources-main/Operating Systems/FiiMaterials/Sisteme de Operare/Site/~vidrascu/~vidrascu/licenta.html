Lucrări de diplomă

Domeniul general: Sisteme distribuite

Coordonez lucrări de licență cu următoarele tematici:

  • Sisteme de operare
          - administrarea sistemelor de operare (Linux, Windows, mobile OSes)
          - programarea sistemelor de operare (Linux, Windows, mobile OSes)
  • Procesare paralelă și distribuită
          - calcule pe clustere de calculatoare, grid computing, cloud computing
  • Tehnici de modelare și verificare a sistemelor distribuite
          - rețele Petri



Iată și câteva posibile subiecte pentru lucrări de diplomă:

  • [mai multe proiecte] Rezolvare paralelă a problemei dificile X
    Descriere: Se cere scrierea unei aplicații pentru rezolvarea uneia dintre problemele dificile (d.p.d.v. computațional) întâlnite la disciplinele studiate pe parcursul facultății, aplicație ce va fi optimizată pentru execuția pe un cluster de calculatoare (sau pe un grid / într-un cloud).
  • [mai multe proiecte] Realizarea de aplicații pentru platforme mobile
    Descriere: Se cere studierea framework-urilor și realizarea unei aplicații software demonstrative pentru oricare dintre platformele mobile actuale (i.e. Android, iOS).

Observații:

  1. Toate programele realizate vor fi open-source (i.e., scrise sub licență GNU GPL).
  2. Sunt binevenite din partea dumneavoastră și alte propuneri de proiecte, pe lângă cele din lista de mai sus, cu condiția să aibă un caracter original și un grad de dificultate corespunzător acestui nivel.

Dacă vă interesează vreunul dintre proiectele de mai sus sau doriți să veniți cu o propunere, contactați-mă direct pentru discuții.
