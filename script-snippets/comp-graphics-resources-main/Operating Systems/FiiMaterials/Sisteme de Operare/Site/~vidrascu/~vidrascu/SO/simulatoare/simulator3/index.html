Simulator grafic pentru algoritmii studiați la curs

Aplicație: TutorSO, i.e. Tutorial pentru cursul de Sisteme de Operare

Este o aplicație Java realizată în 2010 de un student pentru lucrarea sa de licență.
Notă: aplicația are și unele neajunsuri, atât pe partea de funcționalitate (e.g. unele animații nu sunt configurabile), cât și pe partea de interacțiune cu utilizatorul (e.g. rezoluție "fixă", neadaptabilă la diferitele dimensiuni ale diverselor dispozive existente).

Puteți încerca acest simulator pentru a vă fixa sau pentru a aprofunda cunoștințele legate de subiectele abordate la curs.

Descărcați de aici arhiva cu aplicația.   În arhivă veți găsi și un fișier txt cu rol de ghid de utilizare.
Cerințe: necesită un mediu Java instalat pe calculatorul local.
