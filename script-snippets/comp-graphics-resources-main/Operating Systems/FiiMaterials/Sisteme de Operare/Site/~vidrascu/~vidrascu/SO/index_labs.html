Observație: prezentările în format PDF cu lecțiile de programare din tabelul de mai sus conțin hiper-legături http către fișierele sursă C cu exemplele de programe demonstrative ilustrate la cursuri.
Programul Acrobat Reader recunoaște aceste hiper-legături; este suficient să dați click pe aceste hiper-legături și veți putea salva local (sau deschide într-un editor) fișierele respective. Dacă lucrați sub Ubuntu (sau orice altă distribuție de Linux), programul instalat implicit pentru vizualizarea fișierelor PDF nu este Acrobat Reader (deoarece nu este open-source), ci programul Evince, sau pot exista alte programe instalate implicit pentru a vizualiza documente PDF. Programul Evince recunoaște și el aceste hiper-legături, dar numai dacă nu sunteți în modul de prezentare full-screen.
Atât sub Windows, cât și sub distribuția de Linux pe care o folosiți, s-ar putea să aveți însă configurat un alt program pentru vizualizat documente PDF, și nu este exclusă posibilitatea ca acel program să nu știe să recunoască hiper-legăturile http. Soluția în acest caz este următoarea: luați de pe ultimul slide al fiecărei prezentări, cu copy-paste, adresa http a directorului specificat, ce conține toate programele demonstrative din acea lecție, și introduceți-o în navigatorul de Internet pe care-l folosiți în mod uzual.
