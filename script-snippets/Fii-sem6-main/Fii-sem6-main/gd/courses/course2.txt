    Types of objectives:
        -capture = chess
        -chase = catch/evade opponent
        -race = reach a goal before other ppl
        -alignment = arrange pieces in a certain pattern
        -rescue / escape = getr a defined unit or piece to safety (XCOM)
        -Forbidden act = get the opponent to break the rules (Twister)
        - Building = build/mentain structures (Cities:Skylines)
        - Exploration:Explore the game area (Read dead redemption)
        -Solution: Solve a puzzle / problem
        - Outwit = gain/use knowledge to defeat opponents (Diplomacy)
